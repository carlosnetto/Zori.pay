API Reference

Authentication Auth Required

Google OAuth + WebAuthn Passkey two-factor authentication flow

POST/auth/google POST/auth/google/callback POST/auth/passkey/challenge POST/auth/passkey/verify POST/auth/refresh POST/auth/logout

Balance Auth Required

Retrieve wallet balances for all supported cryptocurrencies

GET/balance

Receive Auth Required

Get deposit address for receiving cryptocurrency

GET/receive

Send Auth Required

Send cryptocurrency and estimate transaction fees

POST/send POST/send/estimate

Transactions Auth Required

View transaction history from the blockchain

GET/transactions

Profile Auth Required

Retrieve user profile, contact info, and documents

GET/profile

KYC Auth Required

Account opening and identity verification

POST/kyc/open-account-br

Reference Data Public

Static data for forms: countries, currencies, etc.

GET/reference-data
